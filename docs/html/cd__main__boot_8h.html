void load_vdp_regs ( u16 const *  reg_data)
inline

Load multiple values into VDP registers. Data should consist of word sized values, where the upper byte is the register ID (e.g. 80, 81, etc) and the lower byte is the value. Each value will be stored in RAM in VDP_REG_CACH.
