◆ _GA_MEMMODE

#define _GA_MEMMODE   0xA12002

Word RAM Memory Mode / Sub CPU RAM Write Protect.

F E D C B A 9 8 7 6 5 4 3 2 1 0
WP7 WP6 WP5 WP4 WP3 WP2 WP1 WP0 BK1 BK0 MODE DMNA RET
Parameters
WPWrite protect Sub CPU RAM
BKPRG-RAM bank select
MODEWord RAM layout
DMNAMain CPU will not access Word RAM
RETGive Word RAM control to Main CPU

WP Write protect an area of Sub CPU RAM from 0 to 0x1FE00 in increments of 0x200 BK|PRG-RAM bank select for Main CPU access (4M PRG-RAM divided into 1M banks) MODE|Word RAM Mode | Read Only: 0: 2M / 1: 1M/1M DMNA|"Declaration of Main CPU No Access" on Word RAM | Effect depends on MODE bit:

  MODE = 0 (2M):
   Write: 0: N/A / 1: Return Word RAM to Sub CPU
   Read: 0: Return Word RAM to Sub CPU In Progress
         1: Return Word RAM to Sub CPU Completed
  MODE = 1 (1M/1M):
   Write: 0: N/A / 1: Request 1M bank swap
   Read: 0: Bank swap In Progress
         1: Bank swap Completed

RET|Return Word RAM to Main CPU Effect depends on MODE bit: MODE = 0 (2M): Read Only: 0: Return Word RAM to Main CPU In Progress 1: Return Word RAM to Main CPU Completed MODE = 1 (1M/1M): Read Only: 0: Word RAM Bank 0 attached to Main CPU, Bank 1 to Sub CPU 1: Word RAM Bank 0 attached to Sub CPU, Bank 1 to Main CPU

◆ _GA_RESET

#define _GA_RESET   0xA12000

Sub CPU Reset / Bus Request.

F E D C B A 9 8 7 6 5 4 3 2 1 0
IEN2 IFL2 SBRQ SRES
Parameters
SRESSub CPU reset

W: 0: Reset / 1: Run
R: 0: Reset in progress / 1: Running

Parameters
SBRQSub CPU bus access request

W: 0: Cancel / 1: Request
R: 0: Sub CPU running / 1: Acknowledge

Parameters
IFL2Send INT2 to Sub CPU

W: 0: Not used / 1: Send INT 2 to Sub CPU
R: 0: INT 2 in progress / 1: INT 2 not occurred yet

Parameters
IEN2Mask state of INT2 on Sub CPU

0: Masked / 1: Enabled

Note
Only BTST bitwise operation allowed on this register
