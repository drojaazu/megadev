◆ CDBPAUSE

#define CDBPAUSE   0x0084

Sets the time that the drive spins down from pause to standby.

\group Misc

Warning
BREAK: d0-d1/a0-a1
Parameters
[in]D1.wPause time (in ticks of 1/75 second)

Normal ranges for this delay time are 0x1194 - 0xFFFE. A delay of 0xFFFF prevents the drive from stopping, but can damage the drive if used improperly.

◆ CDBTOCWRITE

#define CDBTOCWRITE   0x0082

Writes data to disc TOC stored in memory.

\group Misc

Warning
BREAK: d0-d1/a0-a1
Parameters
[in]A0.lPointer to TOC data

A TOC entry is four bytes long, where the first three bytes are the BCD timecode of the start of the track and the lowest byte is the track number. The most significant bit of the frame byte of the timecode indicates whether the track is CD-ROM or CD-DA. Please refer to the BIOS manual for more information.

◆ DRVINIT

#define DRVINIT   0x0010

Closes the disk tray and reads the TOC from the CD.

\group Drive

Warning
BREAK: d0-d1/a1
Parameters
[in]A0.lPointer to initilization parameters

Takes a pointer to two bytes (initialization params): byte 1 - track number from which to read TOC (normally 0x01); if bit 7 of this value is set, BIOS will start to play the first track automatically byte 2 - last track to read (0xff will read all tracks)

Pauses for 2 seconds after reading the TOC. Waits for a DRVOPEN request if there is no disk in the drive.

◆ FDRCHG

#define FDRCHG   0x0086

Ramps the audio volume from its current level to a new level at the requested rate.

\group Fader

Warning
BREAK: d0-d1/a0-a1
Parameters
[in]D1.lVolume & Ramp

The input is two 16 bit values High word: new volume (min 0x0000, max 0x0400) Low word: change rate in steps per vblank 0x0001 - slow 0x0200 - fast 0x0400 - immediate

Note
There is a delay of up to 13ms before the volume begins to change.

◆ FDRSET

#define FDRSET   0x0085

Sets the audio volume.

\group Fader

Warning
BREAK: d0-d1/a0-a1
Parameters
[in]D1.wVolume

Setting the upper bit of the value changes the master volume 16 bit volume (0x0000 to 0x0400) 16 bit master volume (0x8000 to 0x8400)

Note
The master volume sets a maximum level which the volume level canot exceed.
There is a delay of up to 13ms before the volume begins to change and another 23ms for the new volume level to take effect.
